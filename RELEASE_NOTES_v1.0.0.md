# v1.0.0 Release Notes

**发布日期**
- 2026-02-03

**新增特性**
- 自动发现并更新 IKUUU 新域名（从登录页与脚本文本中提取）
- 基于历史成功率/最近成功等策略智能排序域名尝试顺序
- 支持邮件探测最新地址与提醒机制（SMTP/IMAP）
- GeeTest v4 验证码解析与 CapSolver/AntiCaptcha 解码支持
- 运行健康检查摘要，适合 cron 定时长期运行

**改进**
- 配置模板与 README 文档完善，支持更清晰的配置与部署流程
- 增加测试与 CI 工作流（Python 3.8）

**兼容性**
- Python 3.8+ 运行
- 依赖库仅 `requests`

**已知限制**
- 运行日志可能包含敏感信息，请确保日志文件不上传
- crontab 环境变量需显式配置
